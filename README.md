# keyboardcat
An esoteric programming language designed to look like a cat walked over the keyboard.
## Example programs
#### Hello World
```
$"Hello, World"
```
#### Truth machine
```
.Pn48{-}p($#)$#
```
#### FizzBuzz
```
#0$##1(>##1<#P#P#Pn3{%}Pn0{=}p[$"Fizz">##0<#]pP#P#Pn5{%}Pn0{=}p[$"Buzz">##0<#]p>#[<#$#>#]<#P#Pn1{+}p>#>#>##10$#c<#<#<#)
```

